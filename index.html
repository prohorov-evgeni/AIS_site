Лабораторная работа №2: Ветвление в Git

Задание на лабораторную работу

Подготовить каркас веб-страницы при помощи популярных фреймворков быстрой разработки в условиях командной разработки с использованием системы контроля версий Git.

Требования к работе

  • Заголовок страницы: включает в себя текст названия лабораторной работы;
  • Содержимое страницы: 2-колоночная (адаптивная) вёрстка. В левой колонке — навигация, справа — текст задания лабораторной работы;
  • Подвал страницы: включает в себя текст авторов лабораторной работы. Имя главного р азработчика выделить полужирным.
  • Использовать HTML5 (HTML 4.01 Strict), CSS3 (CSS2), JavaScript (jQuery, Modernizr) и Twitter Bootstrap;
  • Использовать успешную модель ветвления в Git для ведения командной разработки. Каждый из разработчиков команды должен выполнить не менее четырёхкоммитов;

Результатом выполнения лабораторной работы является демонстрация командой готовой страницы в веб-браузере и подготовленный отчёт от каждого из разработчикав письменном виде, который должен включать в себя:

1. Титульный лист;

2. Теоретические сведения о ветвлении в Git;

3. Вывод команд:

git branch -a
git log --graph --all

4. Список специфичных использованных команд;

5. Вывод.

(с) Удачник по жизни!
